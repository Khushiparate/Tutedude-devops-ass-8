# AWS Terraform Deployment Assignment

## Part 1 - Single EC2
- Flask on port 5000, Express on port 3000
- Single EC2 instance using Terraform

## Part 2 - Separate EC2 Instances  
- Flask EC2 (port 5000) and Express EC2 (port 3000)
- Custom VPC with security groups

## Part 3 - Docker + ECR + ECS + ALB
- Docker images pushed to ECR
- ECS Fargate services
- ALB on port 80 (Flask) and 8080 (Express)

## Tools
- Terraform, AWS, Docker, Flask, Node.js
